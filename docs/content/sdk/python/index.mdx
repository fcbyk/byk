# Python SDK

byk 插件开发的 Python SDK，零依赖。

## 特性

- **零依赖** — 模块顶层仅使用 Python 标准库，第三方依赖由 `@requires` 装饰器按需守卫
- **插件上下文** — `plugin()` 一行获取持久化、日志和目录
- **工具集** — 网络、CLI 输出、Web 辅助函数

## 安装

```bash
pip install byksdk
```

## 快速开始

```python
from byksdk import plugin

ctx = plugin("my_plugin")
ctx.logger.info("启动成功")
ctx.state().set("initialized", True)
```

## 目录结构

```tree
~/.byk/
├── plugins/                 # 插件沙箱
│   └── my_plugin/
│       ├── state.json       # ctx.state()
│       └── my_plugin.log    # ctx.logger
├── state/                   # 全局持久化
│   └── plugins.state.json   # ctx.app.store()
├── logs/
│   └── plugins.log          # ctx.app.logger
├── runtime/
└── cache/
```

## 依赖模型

byksdk 遵循「谁使用谁负责依赖」原则：

- 模块顶层零第三方依赖，全部为 Python 标准库
- 需要第三方库的函数使用 `@requires` 装饰器守卫，缺失时抛出清晰的 `ImportError`

| 使用了... | 需要在 pyproject.toml 加 |
|---|---|
| `create_spa` / `R` / `get_client_ip` | `flask>=2.0` |
| `copy_to_clipboard` | `pyperclip>=1.9.0` |
| `get_private_networks` | `psutil>=5.9.0` |
| `check_port` / `colored_key_value` 等 | `click>=8.0.0` |
| 其他 API | 无需额外依赖 |
