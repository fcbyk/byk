# pysdk

byk python 插件开发套件，以两个独立 `.py` 文件分发。无框架约束，纯函数调用，按需取用。

## 安装

```bash
byk add fcbyk/byk/pysdk
```

将以下两个文件下载到当前工作目录：

| 文件 | 内容 | 
|------|------|
| `byksdk.py` | 插件上下文、StateStore、路径常量、日志 |
| `utils.py` | 网络检测、CLI 输出、Web 辅助函数 |

这就是普通的 `.py` 文件，**用不到的函数可以直接删除**，没有任何隐藏依赖。
