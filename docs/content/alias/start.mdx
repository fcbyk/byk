# 命令别名

通过别名快速运行终端命令

## 快速入门

<Steps>

### 创建配置文件

在当前目录下创建任意 `hello.byk.json`：

```json
{
  "hello": "echo hello world"
}
```

### 通过别名运行

```bash
byk hello
```
